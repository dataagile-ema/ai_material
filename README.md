# AI Material

En samling av demos och lärandematerial relaterat till AI och maskininlärning.

## Demos

### ML/LLM Learning
Interaktiv visualisering av hur en språkmodell lär sig genom träning.

**Plats:** `demos/ml-llm-learning/`

### AI Coding Trends
Översikt och analys av AI-verktyg för kodning och systemutveckling.

**Plats:** `demos/ai-coding-trends/`

## Syfte

Detta repository innehåller utbildningsmaterial och demonstrationer för att förstå och utforska AI-koncept och verktyg.

## Användning

Varje demo är fristående och kan öppnas direkt i en webbläsare. De flesta är enkla HTML-filer som inte kräver någon byggprocess.
