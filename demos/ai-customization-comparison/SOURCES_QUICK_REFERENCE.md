## Kritiska claims per kategori

### Projektinstruktioner
- **Claude:** `.claude/CLAUDE.md`, `~/.claude/CLAUDE.md`, @imports, /memory
- **Cline:** `.clinerules/`, `~/Documents/Cline/Rules`, AI-redigerbar
- **Copilot:** `.github/copilot-instructions.md`, applyTo glob patterns
- **Codex:** AGENTS.md, multi-level hierarchy, 32 KiB limit

### Specialiserade agenter
- **Claude:** Native subagents, separata contexts
- **Cline:** Code/Architect/Ask/Debug modes
- **Copilot:** Agent Mode, Coding Agent (GitHub Actions), @workspace/@github/@terminal
- **Codex:** Ingen native support (community MCP)

### Skills
- **Claude:** YAML frontmatter, automatisk discovery
- **Cline:** MCP marketplace, autonomous MCP creation
- **Copilot:** Agent Skills, max 64/1024 chars, Skillsets (5 max)
- **Codex:** Custom prompts ($1-$9, $FILE), MCP integration

### Hooks
- **Claude:** PreToolUse/PostToolUse, pattern matchers
- **Cline:** 4 events (TaskStart, PreToolUse, PostToolUse, UserPromptSubmit)
- **Copilot:** Webhooks (begränsad, inte i Chat)
- **Codex:** Endast notification hook

### MCP
- **Claude:** STDIO/HTTP, @server:protocol://path
- **Cline:** One-click marketplace, autonomous creation
- **Copilot:** VS Code 1.102+, GitHub MCP Registry
- **Codex:** Codex kan vara MCP server, Agents SDK

### Commands
- **Claude:** $ARGUMENTS/$1/$2, @path, !command
- **Cline:** Built-in modes, .clinerules/workflows/
- **Copilot:** /explain/fix/tests/doc/new/search, @mentions
- **Codex:** /model/status/diff/review/compact/undo, /prompts:<name>

### Output
- **Claude:** Output styles, överskriver system prompt
- **Cline:** UI customization (sidebar, theme, font)
- **Copilot:** Natural language, Markdown only
- **Codex:** Reasoning controls, --json flag, Ctrl+T

### Plugins
- **Claude:** plugin.json manifest, marketplace
- **Cline:** Inget native (MCP istället)
- **Copilot:** Skillsets/Agents/Extensions/MCP, 1,400+ connectors
- **Codex:** Inget native (MCP istället)

### Settings
- **Claude:** local→projekt→användare→enterprise hierarchy
- **Cline:** Manual/Auto-Approve/Maximum Requests, system keychain
- **Copilot:** Personal/Organization/Workspace (2026)
- **Codex:** Approval policies, sandbox modes, requirements.toml
